# Three.js 3D Scene Manipulation

Aplicación web que muestra diferentes ejemplos de manipulación de escenas 3D utilizando la librería **Three.js**.  
El proyecto incluye múltiples demos interactivas que permiten explorar distintos tipos de controles y navegación dentro de escenarios tridimensionales.

## Descripción

Esta aplicación fue desarrollada como parte de una práctica académica de la materia **Desarrollo de soluciones en ambientes virtuales**.  
El objetivo del proyecto es demostrar el uso de **Three.js** para la creación de escenas 3D interactivas dentro del navegador, utilizando diferentes tipos de controles de cámara y manipulación de objetos.

La aplicación contiene varias demos que pueden visualizarse desde una interfaz principal con **Bootstrap 5**, la cual incluye un **navbar de navegación** y un **footer con los datos del desarrollador**.

## Demos incluidas

### Minecraft Geometry
Escenario inspirado en el estilo de **Minecraft**, donde se utilizan geometrías simples para crear un mundo basado en bloques.

Características:

- Uso de texturas
- Generación de bloques
- Escena tridimensional interactiva

---

### Map Controls
Demo que permite controlar la cámara de manera similar a aplicaciones como **Google Maps**.

Características:

- Movimiento horizontal y vertical
- Zoom
- Navegación libre por el escenario

---

### Orbit Controls
Permite rotar la cámara alrededor de los objetos de la escena.

Características:

- Rotación alrededor del objeto
- Zoom con scroll
- Movimiento suave de cámara

---

### PointerLock Controls
Modo de navegación tipo **videojuego FPS (First Person Shooter)**.

Características:

- Movimiento con teclado
- Control de cámara con el mouse
- Experiencia de exploración en primera persona
